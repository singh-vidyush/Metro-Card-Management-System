# Metro-Card-Management-System

A console-based Java application that simulates a metro smart card system.

## Features
- Create and recharge smart cards
- Metro entry and exit with fare deduction
- Balance validation and custom exceptions
- Journey and transaction tracking
- Menu-driven console interface

## Tech Stack
- Java (Core Java, OOP)
- In-memory storage
- No frameworks

## How to Run
Run `Main.java` and follow the on-screen menu.
